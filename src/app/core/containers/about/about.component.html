Purpose

The purpose of this application is to explore the design of an Angular ngRx Signal Store state management with a focus on maintaining sustainable development velocity and facilitating the seamless addition of new features over time like zoneless.

This application provides high-level recommendations for designing a well-structured Angular architecture based on best practices in NgRx Signal Store state management, with a focus on:

- Signal-based Component Architecture.
- Smart and Dumb components pattern.
- Reactive state management (ngrx signal store).
- Zoneless change detections.
 Description

This application is built using Angular-cli.19 and ngrx signal store state management. The application provides the following features:

- Multilingual Support (Internationalization in 3 Languages).
- Apartment search Page.
- Filter Apartments by City and Borough, and load more apartments via scroll.
- Apartment overview page.
- Saving Apartment as Favorites and Sync with Storage.
- Apartment favorites page.

Extras:

- Dynamically load a configuration file before the app starts.
- Register and use custom SVG icons within the application
- Implement internationalization using a third-party library.
- Integrate Infinite Scroll to a view listing.
- Implement Global Error Handling in the web application.
- Setting Page Titles Natively With The Angular Router.
